## 🌍 Why This Exists

Most AI prompt libraries are built for Silicon Valley problems.

This one is built for **African entrepreneurs** — people running logistics companies, tourism agencies, tech startups, and creative studios in cities like Kigali, Lagos, Nairobi, and Accra.

These prompts are practical, direct, and field-tested.

## ⚡ Quick Start

Pick a category → copy the prompt → replace the `[brackets]` with your context → paste into ChatGPT, Claude, or any LLM.
